Voetbal.nl - JSON feeds

Lange tijd heb ik, Jelle, er op gewacht, dat de site voetbal.nl een API beschikbaar zou stellen om uitslagen en programma's van een club in te kunnen zien. Helaas is dit er nooit van gekomen en heb ik zelf maar de stoute schoenen aangetrokken. Bij deze een simpele REST API om op basis van je club-id van de KNVB, het programma en de uitslagen op te halen.

Hieronder zie je een voorbeeld voor s.v. VVZ '49 uit Soest:

http://voetbal.gnojed.nl/BBCC89Q/all
http://voetbal.gnojed.nl/BBCC89Q/programma
http://voetbal.gnojed.nl/BBCC89Q/uitslagen

Programma en uitslagen worden opgehaald...
